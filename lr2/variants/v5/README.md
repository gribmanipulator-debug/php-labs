# Варіант 5 — Лабораторна робота №2

Розв'язок варіанту 5 лабораторної роботи №2 (рядкові функції, масиви, форми, калькулятор).

## Особливості варіанту 5

| Завдання     | Деталі варіанту                                                |
| ------------ | -------------------------------------------------------------- |
| 1 Replace    | Текст про українську народну пісню, заміна "ь" → "й"           |
| 2 Sort       | `sort` — алфавітний порядок міст Полтавської області           |
| 3 Filename   | Шлях `D:\Music\Collection\melody.mp3`                          |
| 4 Dates      | 01-01-2024 → 01-01-2026, тижні + дні                           |
| 5 Password   | 3 паролі довжиною 16, вибір найсильнішого (бальна система 0-5) |
| 6 Duplicates | Масив [2,8,15,2,4,15,8,19,4,11] → дублікати                    |
| 7 Animals    | Склади "зі мо ну ра ти хо ві ді бу", 3 імені по 4 склади       |
| 8 Arrays     | `createArray()` 3-7 елементів 10-20, merge + unique + sort     |
| 9 Assoc      | 7 людей (ім'я → вік), ksort / asort                            |
| 10 Form      | Логін `tryzub_olha`, 10 міст України, сесія + cookie           |
| 11 Calc      | X=1, Y=5: sin, cos, tg, my_tg, x^y, x!                         |

## Запуск

```bash
php -S localhost:8000
```

Відкрийте: `http://localhost:8000/lr2/variants/v5/`
